# Нет доступа к файлам/папкам

**Симптомы:**  
- «Отказано в доступе» к папке или сетевому диску  
- Права администратора не помогают

**Пошаговая диагностика:**

1. Запустить проводник от имени администратора

2. Проверить права (Правая кнопка -> Свойства -> Безопасность -> Изменить):
   - Добавить свою учётную запись
   - Поставить галочки «Полный доступ»

3. Сбросить права через cmd (от администратора):
   
   takeown /F "C:[Путь к папке]" /R /D Yes  - команда для получения прав владельца на папку и её содержимое
   icacls "C:[Путь к папке]" /grant Administrators:F /T - команда для предоставления группе «Администраторы» полных прав доступа к указанной папке и всем её подпапкам и файлам.
